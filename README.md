# Airbnb_dashboard link- https://drive.google.com/file/d/1ReTw7pM_UfKBqqkZYYBMUP0-oUn6P-Ed/view?usp=drive_link
This interactive dashboard delivers deep insights into short-term rental performance, focusing on pricing patterns, seasonal demand, customer trust indicators, and revenue distribution. It enables users to explore how different factors impact listing performance and occupancy trends. This project focuses on real analytical thinking like data modeling, DAX, Pareto analysis, seasonality, pricing, and trust insights.

Purpose
To support data-driven decision-making by analyzing pricing strategies, seasonal booking behavior, and host performance while identifying key drivers of revenue and customer trust.

Tech Stack
Power BI – Data visualization & dashboard development
SQL – Data extraction and transformation
Excel / CSV – Data preprocessing
DAX – Measures, KPIs, Pareto analysis, and advanced calculations

Data Source
Dataset sourced from Maven analytics (Airbnb listings, pricing, and reviews dataset)

Key Features / Highlights

* KPI cards highlighting total revenue, average price, occupancy rate, and number of listings
* Pareto analysis (80/20 rule) to identify top-performing listings contributing to the majority of revenue
* Seasonality analysis to uncover peak booking periods and demand fluctuations
* Pricing insights comparing average price across locations, property types, and room categories
* Trust & review analysis based on ratings, number of reviews, and host credibility indicators
* Interactive slicers (location, property type, room type, season) for dynamic filtering
* Geographic visualization to analyze location-based performance and pricing trends
